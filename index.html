Configurar Firebase

Cole as credenciais do seu projeto Firebase para ativar sincronização em tempo real.

1

Criar projeto

Acesse console.firebase.google.com → Adicionar projeto → nome: tre-ma-atas

2

Ativar Firestore

Firestore Database → Criar → Modo de teste → região southamerica-east1

3

Copiar credenciais

Engrenagem → Configurações → Seus aplicativos → App da Web → copie o firebaseConfig

Colar abaixo
